### Cron 定时任务

| 配置 | 值 |
|------|-----|
| 调度 | 每天 09:00 (`0 9 * * *`) |
| 投递渠道 | QQ Bot |
| 当前状态 | ✅ 正常运行 |

### 生成器工作流

```
每天 9:00
  │
  ├── 计算当前是第几天
  ├── 从 overview.md 解析今日学习主题
  ├── 调用对应的内容生成器（知识点 + 实战练习）
  ├── 生成完整的 README.md 文档
  ├── Review 检查（占位符、代码块、表格等）
  ├── 提交 Git 仓库
  └── 投递学习报告到 QQ
```

### 手动使用

```bash
# 生成指定天的学习资料
cd /root/sre_learning
python3 scripts/daily_study_generator.py --day 15

# 批量生成
python3 scripts/batch_generate_days.py --start 1 --end 28
```
